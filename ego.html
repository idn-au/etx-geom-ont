Provenance

This ontology was made within the IDN's ID Catalogue project 2021 - 2023.

Description

This ontology extends the geometry representation Knowledge Graph elements used by GeoSPARQL 1.1 to cater for narrations of location and qualified geometries. Collections of qualified geometries allow for modelling options such as "fuzzy" geometries.

Code Repository

https://github.com/idn-au/ext-geom-ont/

IRI https://w3id.org/idn/def/ego/QualifiedGeometry
Is Defined By Extended Geometries Ontology
Description

A qualified - limited or restricted - Geometry
Example 
PREFIX ego: <https://w3id.org/idn/def/ego/>
PREFIX egor: <https://w3id.org/idn/def/ego/role/>
PREFIX geo: <http://www.opengis.net/ont/geosparql#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

<http://example.com/feature/wagga-wagga>
    a geo:Feature ;
    ego:hasQualifiedGeometry [
        a ego:QualifiedGeometry ;
        ego:hasGeometry [ ... ] ;
        ego:hasGeometryRole egor:BoundaryEstimate ;
        rdfs:comment "This geometry's scope is qualified by it having a role of Boundary Estimate"@en ;
    ] ;

has geometry op

IRI https://w3id.org/idn/def/ego/hasGeometry
Is Defined By Extended Geometries Ontology
Description

A shadow property of geo:hasGeometry with altered domain & range to allow ego:QualifiedGeometry instances to indicate the Geometry objects they qualify
Example 
PREFIX ego: <https://w3id.org/idn/def/ego/>
PREFIX geo: <http://www.opengis.net/ont/geosparql#>
PREFIX rdfs: <http://www.w3.org/2000/01/rdf-schema#>

<http://example.com/feature/wagga-wagga>
    a geo:Feature ;
    ego:hasQualifiedGeometry [
        a ego:QualifiedGeometry ;
        ego:hasGeometry [
            a geo:Geometry ;
            geo:asGeoJSON "..."^^geo:geoJSONLiteral ;
        ] ;
    ] ;
.

Range ego:Geometry

Datatype Properties

as narrative dp

IRI https://w3id.org/idn/def/ego/asNarrative
Is Defined By Extended Geometries Ontology
Description

A narrative description of a Geometry
Example 
PREFIX ego: <https://w3id.org/idn/def/ego/>
PREFIX ex: <http://example.com/>
PREFIX geo: <http://www.opengis.net/ont/geosparql#>
PREFIX sdo: <https://schema.org/>

ex:moreton-island
    a geo:Feature ;
    geo:hasGeometry [
        a geo:Geometry ;
        ego:asNarrative '''On the eastern side of Moreton Bay on the coast of South East Queensland, Australia with the Coral Sea on the east coast of the island. Moreton Island lies 58 kilometres (36 mi) northeast of the Queensland capital, Brisbane.'''@en ;
    ] ;
    sdo:name "Moreton Island"@en ;
    sdo:alternateName "Mulgumpin" ;
.

Description

A thing or set of things helpful in forming a conclusion or judgment about the validity of a ego:QualifiedGeometry
